# SuperCali Roleplay - Close Beta Update Log 🚀

Yo bre! Hari ini kita ngelakuin update besar-besaran untuk Close Beta Build SuperCali. Gak ada lagi update ecek-ecek, ini perubahan serius biar kota kita makin stabil, realistis, dan siap buat pengumuman akbar!

### UPDATE LOGS:

- **[FIX] Spawn Beda Alam & Sky Camera Sync:** Mengatasi glitch nyangkut di dimensi lain (backrooms) saat loading karakter. Ditambah transisi kamera sinematik gaya Trevor GTA V turun dari langit, lengkap dengan auto-unfreeze & collision recovery biar karakter gak beku jadi batu pas mendarat, serta penyesuaian gender model otomatis anti-bug kelamin!
- **[NEW] Low Health Tinnitus & Echo Audio Effect:** Efek dramatis pas darah kritis di bawah 20% HP! Layar bakal ngeblur merah bergetar, detak jantung berdegap kencang, dan suara sekitar mulai menggema sunyi (tinnitus audio scene asli engine GTA V) biar kerasa sensasi sekarat yang super imersif!
- **[NEW] LEO K-9 Companion Unit:** Polisi/LEO sekarang bisa memanggil anjing pelacak K-9 Rottweiler pendamping (`/k9`) dan memberikan perintah serang (`/k9attack`) ke suspek terdekat secara dinamis!
- **[NEW] Wildlife Hunting & Animal Skinning:** Populasi hewan liar (rusa, babi hutan, singa gunung) sekarang berkeliaran di area Chiliad & Paleto. Bisa diburu menggunakan senjata berburu (Musket/Shotgun) dan dikuliti pakai Pisau/Machete untuk mendapatkan daging komoditas! Tanpa blip di map biar berburunya lebih menantang!
- **[NEW] Cinematic Heist Planning Board:** Fitur rapat sinematik `/heistboard` siap digunakan untuk merencanakan robbery akbar bersama komplotan.
- **[FIX] Refactor Target & Core Exports:** Migrasi besar-besaran dari ekspor lawas `qb-target` yang tidak didukung langsung ke ekspor murni native QBox Core (`exports.qbx_core`) & `ox_target` agar performa server tetap ringan.
- **[FIX] Clean Up Massive Console Script Errors:** Membasmi tuntas puluhan error merah di konsol server (Scoping decompiler goto `lbl_33` & `lbl_66` di script VIP robbery creator, limitasi 200 local variables di houserobbery, CircleZone nil kasino, typo SetBlipColor di govdocument, dan error MOTD chat di qbx_core) demi server zero-lag!
